# Multi-Creator Ingestion Handoffs

This directory tracks implementation accountability for the multi-creator ingestion work.

Each task file must include:

- Owner agent and role.
- Scope and files touched.
- Implementation notes.
- Verification commands and outcomes.
- Spec compliance review.
- Code quality review.
- Open risks or follow-up items.

Reviewer findings must stay in the task file until fixed and re-reviewed.
